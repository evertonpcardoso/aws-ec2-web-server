# 🚀 AWS EC2 Web Server

Projeto de criação de servidor web utilizando AWS EC2.

## Tecnologias
- AWS EC2
- Amazon Linux
- Apache (httpd)

## O que foi feito
- Criação de instância EC2
- Configuração de acesso SSH
- Liberação de portas (80/22)
- Instalação de servidor web

## Resultado
Servidor acessível via navegador com IP público.
